1. Access the WebUI
------------------------------------
Stalwart is deployed in a declarative deployment mode.
For more information checkout the docs https://stalw.art/docs/configuration/declarative-deployments

{{- if .Values.ingress.enabled }}

Access Stalwart at the following URL:

  {{- range .Values.ingress.hosts }}
  https://{{ .host }}{{ (index .paths 0).path }}
  {{- end }}
{{- else }}

Ingress is not enabled. Stalwart is not exposed via HTTP/HTTPS.

{{- end }}

2. Configure Stalwart (WebUI or cli)
------------------------------------
- Add first domain (Management > Domains)
- Add TLS certificate (Settings > TLS > Certificates)
- Configure hostname and defaults (Settings > Network > General)
- Optional: Configure Blob Store (Settings > Storage > Blob Store)
- Optional: Configure Search Store (Settings > Storage > Search Store)
- Optional: Configure In-Memory Store (Settings > Storage > In-Memory Store)

Configuration complete. Restart Stalwart or reload config (Management > Actions)

  kubectl delete pods stalwart-0

You can also configure Stalwart via the stalwart-cli.
